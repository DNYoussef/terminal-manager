10. MANUAL TESTING GUIDE
================================================================================

Step 1: Open Application
  - Navigate to: http://localhost:3002
  - Verify: Theme toggle button visible in navigation

Step 2: Test Toggle Cycle
  - Click toggle button once -> Light mode
  - Click toggle button twice -> Dark mode
  - Click toggle button three times -> System mode
  - Verify: Icons and labels change correctly

Step 3: Test Persistence
  - Set theme to Light
  - Reload page (Ctrl+R)
  - Verify: Still in Light mode
  - Repeat for Dark and System modes

Step 4: Test System Preference
  - Set theme to System mode
  - Change OS theme (Windows Settings > Personalization > Colors)
  - Verify: App theme updates automatically

Step 5: Test Accessibility
  - Press Tab key to focus toggle button
  - Press Enter or Space to toggle theme
  - Verify: Theme changes and screen reader announces
